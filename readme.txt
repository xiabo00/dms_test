# Version: 1.0.2
1.软件介绍：该软件主要功能是对硬件设备进行软件初始化，OTA升级，以及软件切换，
目前支持的硬件有LMDC,LMDC-V2,LBB300,LBB400这四种硬件设备，都支持本地操作或者通过AWS操作
然后LMDC6000目前只支持本地操作。

2.打包方式：将代码拉入本地之后，需要先执行pyrcc5 images.qrc -o images_rc.py命令
这个步骤是为了将图片编译，然后执行打包命令pyinstaller --onefile --windowed --name DMS main.py
生成的DMS.exe在dist目录下，然后还需要将Shadow这个目录放在和DMS.exe同个目录下，因为后续初始化模式的时候会用到。

3.具体操作方式：
注意：目前只支持3.0.5版本以及以后的新版本。
1.打开DMS软件后，第一步是连接设备，LOCAL就是你本机的IP地址，DEVICE就是你需要连接的设备的IP地址。
2.连接成功后，会自动获取SN码，选择设备类型进行匹配，必须和SN码中的设备类型成功匹配才行。
3.匹配成功之后，就可以选择你想要操作的模式，INIT就是初始化模式，OTA就升级模式，SWITCH就是切换软件模式
然后还有一个Mode Switch按钮，这个按钮是用来切换你选择模式的，比如你点击的是INIT模式，此时你不想操作这个模式了
就可以点击这个按钮进行模式切换。
4.选好模式之后，会有一个弹窗弹出，这个弹出是用来配置设备shadow的，你需要根据实际情况进行配置，这一步是必须要的，
然后配置shadow的时候你可以通过图片来选择你想要使用的串口。
5.配置好shadow之后，选择你想要的软件类型，选择操作方式，如果是Local的话就是通过本地来操作，如果是S3的话就是通过AWS来操作。
6.选择好了之后，就需要获取当前版本了，如果你是INIT模式的话，此时是不需要获取的，但如果是其它的模式就需要获取到当前版本。
7.获取到当前版本之后，如果此时是Local模式，就需要你从本地浏览软件包进行上传，然后点击start开始按钮就好了，
如果是S3模式的，选择好你想操作的软件点击Start就可以了。
8.LOG下载，当你的设备连接成功，并且匹配成功之后，就可以下载log文件了，点击下载后会有一个选择时间的弹窗，选择好时间后，会出现一个将log下载到本地的哪里的弹出，
这个弹窗中可以选择你想下载的log文件。
